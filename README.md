<h4>Installation</h4>
<pre>
cd &lt;your-project-folder&gt;
git clone https://github.com/RandallWiggins/react-stub .
npm install
webpack --watch
</pre>
<p>
A couple of static files will be created in ./build ready to be delivered by your favorite HTTP server.
